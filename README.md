# Python Job Scraper

This project is a simple web scraper built with Python.

It scrapes job listings from the Fake Python Jobs website and saves the data into a CSV file.

## Features

- Scrape job title
- Scrape company name
- Scrape location
- Scrape job detail URL
- Export data to CSV

## Technologies Used

- Python
- Requests
- BeautifulSoup4
- CSV

## Project URL

https://realpython.github.io/fake-jobs/

## How to Run

```bash
pip install requests beautifulsoup4
python scraperfakejob.py
```

## Output

Data will be saved to:

```text
jobs.csv
```
